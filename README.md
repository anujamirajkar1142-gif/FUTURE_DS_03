# Marketing Funnel & Conversion Performance Analysis Dashboard

## Internship Task

Future Interns – Data Science & Analytics Internship

Task 3: Marketing Funnel & Conversion Performance Analysis

## Project Overview

This project analyzes marketing funnel performance using the Bank Marketing Dataset. The dashboard helps identify conversion patterns, customer engagement trends, and key factors influencing successful conversions. The objective is to understand how prospects move through the marketing funnel and discover opportunities to improve campaign effectiveness.

## Tools Used

- Power BI
- DAX
- Power Query
- Data Cleaning & Transformation

## Key KPIs

- Total Contacts: 41K
- Converted Customers: 5K
- Conversion Rate: 11.27%
- Average Age: 40.02 Years

## Dashboard Features

- Marketing Funnel Analysis
- Conversion Rate by Contact Type
- Customer Conversion by Marital Status
- Conversion Trend by Month
- Conversion Distribution by Job Role
- Job & Education Conversion Heatmap
- Monthly Conversion Performance Analysis
- Interactive Filters & Drill-down Insights

## Key Insights

- Cellular contact campaigns achieve significantly higher conversion rates than telephone campaigns.
- Married customers contribute the highest share of successful conversions.
- Administrative and Technician job categories generate the most conversions.
- Conversion performance varies across education levels and occupations.
- Customer conversions show seasonal trends across different months.

## Recommendations

- Prioritize cellular-based marketing campaigns.
- Focus marketing efforts on high-converting customer segments.
- Create targeted campaigns based on occupation and education profiles.
- Optimize campaign timing during high-performing months.
- Improve engagement strategies for low-converting customer groups.
